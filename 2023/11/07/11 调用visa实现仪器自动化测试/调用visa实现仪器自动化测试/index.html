1、测试环境准备

1）测试环境应用基础

a.SCPI指令集

可编程仪器标准命令，定义了一套用于控制可编程测试测量仪器的标准语法和命令。命令主要分为Set（控制）和Query（查询）两种。

b. GPIB

通用接口总线是一种设备和计算机连接的总线。大多数台式仪器是通过GPIB线以及GPIB接口与电脑相连，系统中最多不能超过15台仪器。

如果是台式机或者笔记本作为主控机，需要买一个USB转GPIB的线。

c.Visa

Virtual instrument software architecture，虚拟仪器软件结构。可以使用网线连接仪器，通过网络地址来程控仪器，也可以通过GPIB线，使用端口程控仪器。

2）安装软件环境

a.安装Python： http://www.zhanshaoyi.com/16338.html

注意一定要勾选Add Python 3.9 to PATH，不然后面使用pip指令要重新添加环境变量，比较麻烦

b.安装vscode

c.安装Python库

值cmd命令行执行安装脚本：pip install pyvisa ，根据需求安装所需的Python库就可以了

d.安装ipop

软件下载： /rnd/zhanghao/ipop_4.1.exe

IPOP的主要目的是绑定虚拟ip，也可以直接修改ipv4的地址，但是个人觉得ipop用起来更加方便推荐ipop，绑定虚拟ip的主要目的和仪器进行连接，注意要选择和实际网口对应的网卡端口，绑定和仪器相同的网段

通过ipconfig 即可查看是否绑定成功

如果可以pin 通就可以进行下一步操作
